# test-ritm

## Тестовое задание

### Для запуска 
```
git clone https://github.com/remark-ekz/test-ritm.git
```
### Создать и активировать окружение 

```
python -m venv venv
```

```
.venv/Scripts/activate
```

### Установка зависимостей

```
pip install -r requaremet.txt
```

### Прогон тестов

```
pytest -s -vv
```

### Всех тестов

```
pytest -s -vv --runslow
```
